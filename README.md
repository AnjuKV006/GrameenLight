### Smart Electricity Monitoring & Complaint Management System

GrameenLight is an advanced Android-based smart monitoring application designed to improve electricity infrastructure management in rural and urban areas. The application enables citizens to report electrical issues in real time while helping authorities monitor, track, and resolve complaints efficiently through a modern digital platform.

The system focuses on improving public safety, reducing manual complaint handling, and enabling faster response to critical electrical problems such as damaged poles, wire cuts, fire risks, and power failures.

---

# 🚀 Key Features

## 📢 Smart Complaint Reporting
Users can easily submit electricity-related complaints through a simple and interactive interface.

### Features Included:
- Pole ID submission
- Phone number collection
- Problem description entry
- Image upload support
- Automatic GPS location detection
- Real-time complaint storage using Firebase

---

# 📊 Interactive Smart Dashboard

The dashboard provides a professional overview of all complaint activities with live analytics.

### Dashboard Modules:
- Total Complaints
- Pending Complaints
- Resolved Complaints
- Resolution Percentage
- Live Google Maps Integration
- Real-time Monitoring Interface

The dashboard is designed with a modern dark-themed UI inspired by futuristic smart-city applications.

---

# 🗺 Pole Status Monitoring System

The Pole Status module allows administrators and users to monitor the current condition of electricity poles.

### Information Displayed:
- Pole ID
- Registered Phone Number
- Reported Problem
- Complaint Status
  - 🟢 Solved
  - 🔴 Not Solved

All data updates automatically in real time using Firebase Firestore listeners.

---

# 🚨 Emergency Alert Management

The Emergency Alert System identifies and highlights dangerous electrical situations that require immediate attention.

### Emergency Cases Detected:
- Broken Electric Pole
- Wire Cut
- Fire Risk
- Electric Shock
- Spark Detection
- High-Risk Electrical Damage

Critical complaints are visually highlighted for rapid response and improved safety management.
